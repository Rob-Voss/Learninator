insert(item)

Adds a new Item to the node. If the node already has sub nodes, the item gets pushed down one level. If the item does not fit into the sub nodes, it gets saved in the "children"-array. If the maxChildren limit is exceeded after inserting the item, the node gets divided and all items inside the "children"-array get pushed down to the new sub nodes.
