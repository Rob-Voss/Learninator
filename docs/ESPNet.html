Class: ESPNet

ESPNet

new ESPNet(nSp, nInput, nHidden, genes) → {ESPNet}

A variant of ESP network implemented population of N sub neural nets, each to be co-evolved by ESPTrainer fully recurrent. Outputs of each sub nn is also the input of all other sub nn's and itself. inputs should be order of ~ -10 to +10, and expect output to be similar magnitude. user can grab outputs of the the N sub networks and use them to accomplish some task for training
Parameters:
Name Type Description
nSp Number Number of sub populations (ie, 4)
nInput Number Number of real inputs to the system (ie, 2). so actual number of input is Niput + nSp
nHidden Number Number of hidden neurons in each sub population (ie, 16)
genes Array (optional) array of nSp genes (floatArrays) to initialise the network (pretrained)
